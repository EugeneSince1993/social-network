Каждый компонент (функция) вызывается всегда с параметрами. Самый главный параметр - props. props - объект.
props всегда передается в качестве параметра функции-компонента. Если мы пишем, например, "<Header />", тогда передается пустой объект props.
Мы можем добавить к тегу компонента атрибуты с произвольным названием, и тогда React наполнит объект props парами ключ-значение, соответствующими названию атрибута и его значения:
<Header name='Dima K' age='30' />

{
  name: 'Dima K',
  age: '30'
}

JSX-код пишется только внутри команды "return(...);", вне этой команды - обычный JS.

Вместо console.log(...); можно использовать debugger;
props - одна из главных тем в React.

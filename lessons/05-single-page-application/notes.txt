Single page application - это маленькая полупустая HTML-страница, в которую помимо HTML приходит (с сервера) много JavaScript кода.
JavaScript перерисовывает данные на странице.
Благодаря SPA трафик уменьшается, время загрузки страниц сокращается, и мы уменьшаем нагрузку на сервер.
Объекты JSON занимают мало места.
При работе SPA сервер выступает в роли API.
В SPA очень много логики.
От клиента к серверу идут AJAX-запросы.
В json данные с сервера приходят и для SPA, и для мобильного приложения.
"JSON" - это универсальный формат.
В итоге, SPA приносит финансовую экономию за счет экономии ресурсов сервера.

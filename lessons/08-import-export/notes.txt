modules AMD, require.js
Путь к файлу относительный
./ - папка, в которой я нахожусь
./../ - папка уровнем выше
".js" можно не ставить в пути файла

import openGoogle from './../google.js';

на первых порах мы всегда ставим ./ в начале пути файла.

export default openGoogle;

import React, { Component } from 'react';
react - это название модуля (пакета, библиотеки) в "node_modules".

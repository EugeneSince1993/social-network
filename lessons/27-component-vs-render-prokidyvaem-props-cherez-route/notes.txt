react-router (версия до 6)
В Route в атрибуте component передается компонент (функция).
<Route path='/dialogs' component={Dialogs} />
У Route есть атрибут render. Точнее, в props Route ожидает получить render. Атрибуты component и render - взаимоисключающие. В render передается анонимная функция.
<Route path='/dialogs' render={ () => {  } } />
<Route path='/dialogs' render={ () => { <Dialogs  /> } } />

render работает быстрее component

В react-router v6 вариант с атрибутом render не работает.

Компонент - функция, которая возвращает JSX.

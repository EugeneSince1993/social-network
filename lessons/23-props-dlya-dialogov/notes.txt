Мы инкапсулируем (скрываем) детали в компоненты.
Рефакторинг - улучшение кода.
Если у нас есть компоненты, которые используются только внутри одного родительского компонента, тогда код этих компонентов можно держать в файле родительского компонента. Хотя вообще это плохо.
В общем, обычно надо держать каждый компонент в отдельном файле.
Alt+F1 - выбираем пункт Project View, жмем Enter. Таким образом можно открыть файл в проводнике WebStorm.
Ctrl+Alt+L - выровнять отступы.
Можно кликнуть с Ctrl по существующему классу (CSS) в JSX-коде, и тогда мы перейдем в соответствующий файл CSS.

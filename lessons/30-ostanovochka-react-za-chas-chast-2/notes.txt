Функциональный компонент (презентационная/presentational, без состояния/stateless, тупая/dumb).
Есть еще классовый компонент.
Компонент - функция, которая принимает объект props и возвращает JSX-разметку.
map() - метод массива, преобразующий один массив в другой.
map() принимает в качестве атрибута callback function (arrow function).
UI BLL DAL
React (UI) - главная ответственность - отрисовка интерфейса.
Redux (BLL) - главная ответственность - данные и логика.
index.js - точка входа в приложение
Функция не имеет права работать с глобальными объектами (из соображений безопасности).
Чистая функция
Ответственность компонента Router - следить за адресной строкой браузера.

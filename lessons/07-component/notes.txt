Компонент - это функция, возвращающая разметку (JSX).
Наша стрелочная функция App (в "App.js") - это компонент.
Компонент - это кирпич, из которого строится все.
Компонент - кусок разметки (JSX, HTML).
Наш компонент App - это кусок большой страницы.
===

Function expression

// ES5

var x = function(x, y) {
  return x * y;
};

// ES6

Arrow function

const x = (x, y) => {
  return x * y;
};

const x = (x, y) => x * y;

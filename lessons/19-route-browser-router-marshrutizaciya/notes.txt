Ctrl+Shift+N - найти файл и т.д.
Изначально в CRA не стоит пакета маршрутизации, поэтому его нужно отдельно установить.
npm install OR npm i
npm i react-router-dom --save
--save ставим, чтобы пакет сохранился в зависимостях в "package.json".
Начиная с версии npm 5, "--save" ставить не обязательно, npm сам сохранит пакет в зависимостях.
Если проект на Git обновился и не хочет запускаться, прося установить зависимости, пишем "npm install".
Ctrl+D - продублировать строку.
В импорте компоненты ставятся в фигурных скобках, если компонент(-ы) импортируется не по умолчанию.
В react-router-dom ver 6 в компоненте <Router /> вместо свойства component надо использовать element. А также надо в значении свойства element писать компонент в виде тега: <Dialogs />
Example (from stackoverflow):
<Route path="/" element={<Home />}></Route>
Компоненты Route ищут совпадения (по путям, адресам), анализируя текущую ссылку в браузере. Если совпадение есть, они рисуют нужный компонент.

Ctrl+Shift+N - поиск файла по проекту.
В компоненты React приходят данные через props из BLL (уровень бизнес логики - Redux), а не с сервера.
UI - это набор файлов (слой, где лежат файлы, код), который отвечает за отрисовку содержимого (внешнего вида).
BLL - это набор файлов (слой, где лежат файлы, код), который отвечает за хранение данных.
Слой - это набор файлов
Акроним SOLID (single responsibility, open–closed, Liskov substitution, interface segregation и dependency inversion), ООП.
Чистая функция - получает данные через props и всегда на основе этих данных возвращает один и тот же результат.

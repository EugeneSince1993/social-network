git clone https://github.com/it-kamasutra/react-way-of-samurai.git 
скопировать репозиторий (папку проекта) на локальный компьютер по заданному пути.

если выполнить команду react-scripts eject, тогда весь список пакетов будет записан в package.json, и тогда React.js нельзя будет обновить до новой версии (18 и т.д.).

Если надо вернуться к самой первой версии проекта (чтобы начать проект с нуля для обучения), в папке проекта ввести команду:
gitk --all&
Открывается графический интерфейс (графическая оболочка git)
Находим commit №7 (07: what component is).
Копируем его идентификатор (SHA1 ID) с помощью Ctrl+Insert
Затем в окне Git Bash папки проекта ввести команду
git checkout номер_коммита
Shift+Insert - вставить из буфера обмена
package-lock.json - удаляем, ибо он мешает переключению на старую версию проекта
commit #7 - 58557

Это был пример.
Мне нужно переключиться на коммит урока 12.

git checkout src/App.js - уничтожить новые изменения в App.js (вернуть к первоначальной версии этого файла в конкретной версии проекта)

git status - проверить состояние репозитория

detach - отделять
В Git GUI - если нажать F5, тогда текущее состояние по коммитам обновится.
